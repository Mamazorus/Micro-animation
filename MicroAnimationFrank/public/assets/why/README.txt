Vidéos de démo des 3 cartes « Pourquoi Frank ? ».

Déposer ici une boucle (~3 s, muette) par fonctionnalité, nommée selon la
clé de la carte. Le .webm est lu en priorité, le .mp4 sert de repli :

  feed.webm    (ou feed.mp4)     -> Feed personnalisé
  install.webm (ou install.mp4)  -> Installation en 1 clic
  verif.webm   (ou verif.mp4)    -> Skills vérifiés

Cadre d'affichage : object-fit cover, ratio ~ largeur de carte x 150-256 px
de haut. Tant qu'un fichier manque, le fond dégradé violet reste affiché.

Recette d'export cohérente avec le projet : .webm VP9. Pour ces démos
(pas de transparence nécessaire) un .mp4 H.264 fait aussi l'affaire.
